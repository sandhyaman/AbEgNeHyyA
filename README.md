# 项目简介

Python在线考试系统源码、文档

## 核心功能

- 学院信息管理
- 班级信息管理
- 考试科目管理
- 学生信息管理
- 教师信息管理
- 校园考试管理

## 适用场景

- 教育机构进行在线考试
- 教师进行考试管理
- 学生参加在线考试

## 技术特点

- 使用Django框架进行后端开发
- 采用MySQL作为数据库存储
- 前端界面基于Vue框架构建
- 提供完整的源码及1.7万字文档
- 支持运行教程，方便用户快速部署
